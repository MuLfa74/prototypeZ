# prototypeZ
Cайт-приложение для поиска teammate'ов (партнеров по команде) в многопользовательских видео-играх. Представляет собой доску с объявлениями/запросами на поиск партнеров по команде, привязанную к определенной видео-игре, игру выбирает сам пользователь. Сайт ставит цель упрощения задачи нахождения единомышленников и начала коммуникации игроков.
